## 🎯 产品特色

### MVP核心功能
- ✅ 基础约定创建与管理
- ✅ 软提醒与硬执行双模式
- ✅ 亲友见证人机制
- ✅ 端侧AI辅助（必备功能）
- ✅ 多端支持（iOS/Android/小程序）

### 目标用户
- 💕 情侣（主要）
- 👨‍👩‍👧 亲子家庭（主要）
- 🎯 自律人群
- 👥 微信群体

### 设计原则
- **简单易用**：降低理解成本，专注核心功能
- **真实关系**：基于亲友关系，非陌生人社交
- **正向激励**：鼓励为主，惩罚为辅
- **隐私保护**：数据安全，本地优先
