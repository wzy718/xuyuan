# 小约定App - 信誉系统设计（MVP简化版）

## 一、核心理念

针对情侣和亲子场景，信誉系统聚焦于"履约能力"展示，而非社交活跃度。见证人必须是亲朋好友，强调真实关系背书。系统设计以简单、透明、正向激励为原则。

## 二、履约分设计

### 分值模型（极简版）

```
履约分 = 基础分(60) + 完成分 - 违约扣分
```

#### 基础参数
- **分值范围**：0-100分
- **初始分值**：60分（及格线）
- **显示方式**：仅显示星级，不显示具体分数
- **计算周期**：滚动30天

### 星级展示系统

#### 星级划分
- ⭐ **一星**（0-39分）：需要努力
- ⭐⭐ **二星**（40-59分）：正在成长  
- ⭐⭐⭐ **三星**（60-79分）：值得信任
- ⭐⭐⭐⭐ **四星**（80-94分）：非常可靠
- ⭐⭐⭐⭐⭐ **五星**（95-100分）：完美履约

#### 展示设计
- 个人主页顶部展示
- 约定详情页参与者旁显示
- 不显示具体分数，降低压力
- 鼠标悬停显示评级说明

### 计分规则（简单透明）

#### 加分项（仅统计最近30天）

| 行为 | 加分值 | 说明 |
|------|--------|------|
| 完成一个约定 | +2分 | 基础完成分 |
| 提前完成 | +1分 | 额外奖励 |
| 连续7天履约 | +5分 | 连续奖励 |
| 收到感谢 | +1分 | 伴侣/孩子点击感谢 |
| 首次完成某类约定 | +3分 | 新手奖励 |

#### 扣分项

| 行为 | 扣分值 | 说明 |
|------|--------|------|
| 未完成约定 | -3分 | 到期未完成 |
| 延期完成 | -1分 | 超期但最终完成 |
| 放弃约定 | -5分 | 主动放弃 |
| 虚假打卡 | -10分 | 被举报证实 |

#### 分数保护机制

- **保底分数**：最低不低于0分
- **滚动计算**：30天自动重算，只看近期表现
- **新手保护**：新用户前3个约定不扣分（学习期）
- **异常保护**：不可抗力导致的失败可申诉免责

## 三、亲友见证人机制

### 见证人定位

#### 核心原则
- 不是平台派遣的陌生人
- 必须是双方共同认识的亲友
- 作用是增加仪式感和约束力
- 不承担专业审核职责

### 见证人邀请

#### 适合的见证人
- **情侣约定**：
  - 共同好友
  - 闺蜜/兄弟
  - 家人长辈
- **亲子约定**：
  - 另一方家长
  - 爷爷奶奶/外公外婆
  - 孩子的老师
- **自律约定**：
  - 室友
  - 同事
  - 健身伙伴

#### 邀请流程
1. 创建约定时选择"添加见证人"
2. 从通讯录选择或输入手机号
3. 发送微信/短信邀请链接
4. 见证人确认接受邀请

### 见证人职责（轻量化）

#### 基础职责
- **见证签署**：确认知晓这个约定存在
- **到期提醒**：收到推送，可一键提醒当事人
- **结果确认**：简单确认完成/未完成状态

#### 不需要承担
- ❌ 复杂的证据审核
- ❌ 专业的完成度判断
- ❌ 长时间在线守候
- ❌ 频繁的互动要求

### 见证人激励

#### 精神激励
- **见证徽章**：
  - 铜章：见证10个约定
  - 银章：见证30个约定
  - 金章：见证100个约定
- **感谢卡片**：
  - 完成后可发送感谢
  - 积累感谢卡片墙
  - 年度见证人表彰

#### 无金钱激励
- 不设置见证费用
- 避免利益关系影响公正性
- 保持纯粹的友情支持

## 四、信任关系构建

### 关系类型标记

```
情侣关系 → 💕 专属图标 + 甜蜜度展示
亲子关系 → 👨‍👧 家庭图标 + 成长值展示  
朋友关系 → 🤝 友谊图标 + 默契度展示
```

### 关系专属数据

#### 情侣甜蜜度
- **数据维度**：
  - 共同完成约定数
  - 最长连续履约天数
  - 互相感谢次数
  - 共同挑战成功率
- **展示方式**：
  - 爱心进度条
  - 甜蜜指数（❤️×N）
  - 专属纪念日标记

#### 亲子成长值
- **数据维度**：
  - 孩子完成任务数
  - 家长陪伴时长
  - 共同进步里程碑
  - 习惯养成天数
- **展示方式**：
  - 成长树可视化
  - 勋章收集册
  - 成长曲线图

#### 朋友默契度
- **数据维度**：
  - 互相见证次数
  - 共同挑战完成数
  - 相互鼓励频率
  - 约定成功率
- **展示方式**：
  - 默契值百分比
  - 友谊等级提升
  - 共同成就墙

### 关系专属权益

#### 解锁机制
- **情侣达到30天**：解锁情侣专属模板
- **亲子达到100个任务**：生成成长相册
- **朋友见证超过20次**：获得"铁哥们"称号

#### 专属功能
- 关系专属约定模板
- 纪念日自动提醒
- 年度关系报告
- 专属成就系统

## 七、简化后的核心价值

### 用户价值
1. **降低理解成本**：星级一目了然，不纠结分数
2. **聚焦真实关系**：亲友见证，不是陌生人评判
3. **强调正向激励**：重在鼓励，轻在惩罚
4. **保护用户隐私**：约定仅参与者可见
5. **避免功利导向**：不涉及金钱，纯粹记录

### 产品价值
1. **降低运营成本**：无需审核见证人资质
2. **减少纠纷风险**：避免金钱纠纷
3. **提高用户粘性**：基于真实关系
4. **简化产品逻辑**：专注核心功能
5. **快速验证市场**：MVP快速迭代
