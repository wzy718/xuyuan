# 分享图片配置指南

## 图片要求

### 尺寸规格
- **推荐比例**: 5:4
- **推荐尺寸**: 500px × 400px 或 1000px × 800px
- **最大文件大小**: 128KB
- **格式**: JPG 或 PNG

### 设计建议

#### 视觉风格
- 禅意中式风格，与小程序整体设计保持一致
- 使用金色、红色等传统吉祥色彩
- 可包含莲花、祥云、佛珠等元素

#### 文案建议
分享图片上可以包含以下文案元素（选择1-2个即可，避免过于拥挤）:

1. **主标题**（必选）:
   - "愿望分析助手"
   - "拜拜 - AI愿望分析"
   - "让愿望更容易实现"

2. **副标题/卖点**（可选）:
   - "AI帮你优化许愿姿势"
   - "找出愿望缺失要素"
   - "科学许愿，事半功倍"

3. **视觉元素**（建议）:
   - 莲花图案（象征纯净、圆满）
   - 祥云图案（象征吉祥）
   - 金色边框或装饰线条
   - 渐变背景（金色到红色，或蓝色到紫色）

#### 参考布局

## 图片放置位置

### 方式一：放在 `frontend/src/assets/` 目录（推荐）

1. 将图片命名为 `share-cover.jpg`
2. 放置在 `frontend/src/assets/` 目录下
3. 代码中已配置路径为 `/assets/share-cover.jpg`

### 方式二：使用在线图片

如果图片较大或需要动态更新，可以使用在线图片：

```typescript
return {
  title: '我刚测了个愿望分析，准到吓人😱',
  path: sharePath,
  imageUrl: 'https://your-cdn.com/share-cover.jpg' // 使用 HTTPS 链接
}
```

## 注意事项

### 图片相关

1. **文件大小**: 确保图片小于 128KB，否则可能无法显示
2. **图片路径**: 
   - 本地图片路径必须是相对于小程序根目录的绝对路径（如 `/assets/share-cover.jpg`）
   - 在线图片必须使用 HTTPS 协议
   - 在线图片的域名需要在小程序后台配置为合法域名
3. **图片内容**: 不能包含违规内容，否则可能被微信拦截

### 标题相关

1. **长度限制**: 标题不宜过长，建议 15-25 字
2. **内容合规**: 不能包含诱导分享、夸大宣传等违规内容
3. **Emoji 使用**: 适度使用 Emoji，不要过多（1-2 个即可）

### 合规要求

根据微信小程序规范，分享内容不得包含：

- ❌ 诱导分享（如"分享即可获得"、"不转不是中国人"等）
- ❌ 夸大宣传（如"100%准确"、"包治百病"等）
- ❌ 违规内容（如涉及迷信、赌博等）
- ✅ 客观描述功能和体验（如"愿望分析助手"、"AI 优化许愿"等）
