# 分享标题随机显示功能

## 功能说明

分享标题现在支持随机显示，每次分享时会从 4 个备选标题中随机选择一个，增加分享内容的多样性和吸引力。

## 当前使用的标题

1. `快来测测你的愿望能不能实现🎯`
2. `愿望没实现？可能是这些原因🔍`
3. `分享一个超准的愿望分析工具🌟`
4. `测了个我许的愿望，结果惊呆了😳`

## 实现方式

### 代码结构

在两个文件中都添加了相同的逻辑：

1. **定义标题数组**：
```typescript
const SHARE_TITLES = [
  '快来测测你的愿望能不能实现🎯',
  '愿望没实现？可能是这些原因🔍',
  '分享一个超准的愿望分析工具🌟',
  '测了个我许的愿望，结果惊呆了😳'
]
```

2. **随机选择函数**：
```typescript
const getRandomShareTitle = () => {
  const randomIndex = Math.floor(Math.random() * SHARE_TITLES.length)
  return SHARE_TITLES[randomIndex]
}
```

3. **在分享配置中使用**：
```typescript
return {
  title: getRandomShareTitle(), // 随机显示分享标题
  path: sharePath,
  imageUrl: '/assets/share-cover.jpg',
  // ...
}
```

## 工作原理

- 每次调用 `useShareAppMessage` 时，`getRandomShareTitle()` 函数会执行
- 函数会从 `SHARE_TITLES` 数组中随机选择一个标题
- 每次分享时显示的标题可能不同，增加多样性

## 如何修改标题

### 添加新标题

在 `SHARE_TITLES` 数组中添加新的标题：

```typescript
const SHARE_TITLES = [
  '快来测测你的愿望能不能实现🎯',
  '愿望没实现？可能是这些原因🔍',
  '分享一个超准的愿望分析工具🌟',
  '测了个我许的愿望，结果惊呆了😳',
  '你的新标题✨' // 添加新标题
]
```

### 修改现有标题

直接修改 `SHARE_TITLES` 数组中对应的标题：

```typescript
const SHARE_TITLES = [
  '快来测测你的愿望能不能实现🎯',
  '愿望没实现？可能是这些原因🔍',
  '分享一个超准的愿望分析工具🌟',
  '修改后的标题😳' // 修改标题
]
```

### 删除标题

从 `SHARE_TITLES` 数组中删除不需要的标题：

```typescript
const SHARE_TITLES = [
  '快来测测你的愿望能不能实现🎯',
  '愿望没实现？可能是这些原因🔍',
  // '分享一个超准的愿望分析工具🌟', // 删除这个标题
  '测了个我许的愿望，结果惊呆了😳'
]
```

**注意**：修改后需要在两个文件中同步更新。

## 优势

1. **增加多样性**：每次分享显示不同的标题，避免用户产生审美疲劳
2. **A/B 测试**：可以同时测试多个标题的效果
3. **提升吸引力**：不同风格的标题可以吸引不同类型的用户
4. **降低风险**：如果某个标题被微信拦截，其他标题仍可正常使用

## 注意事项

1. **同步更新**：修改标题时需要在两个文件中同步更新
2. **标题数量**：建议保持 3-5 个标题，太少缺乏多样性，太多可能影响用户体验
3. **标题质量**：确保所有标题都符合微信小程序规范
4. **测试验证**：修改标题后需要在微信开发者工具中测试分享功能

## 测试方法

1. 在微信开发者工具中运行小程序
2. 进入"愿望分析"页面
3. 输入愿望并点击"开始分析"
4. 分析完成后，多次点击"分享解锁"按钮
5. 观察每次分享时显示的标题是否不同

## 未来优化建议

1. **基于用户画像选择标题**：根据用户的许愿类型选择更相关的标题
2. **基于时间选择标题**：不同时间段显示不同的标题（如工作日 vs 周末）
3. **基于数据优化**：根据分享点击率数据，调整各标题的显示概率
4. **标题配置化**：将标题配置提取到配置文件或后端，方便动态调整
