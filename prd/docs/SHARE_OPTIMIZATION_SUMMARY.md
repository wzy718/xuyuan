# 分享解锁优化总结

## 优化内容

### 1. 分享标题优化

**优化前：**
```
拜拜：愿望分析助手
```

**优化后：**
```
我刚测了个愿望分析，准到吓人😱
```

**优化原则：**
- ✅ **制造好奇心**：使用"准到吓人"等词汇引发好奇
- ✅ **第一人称**：使用"我刚测了"增加真实感和代入感
- ✅ **使用 Emoji**：😱 表情增加视觉吸引力
- ✅ **简短有力**：控制在 20 字以内
- ✅ **符合微信规范**：无诱导分享、无夸大宣传

### 2. 分享图片配置

**新增配置：**
```typescript
imageUrl: '/assets/share-cover.jpg'
```

**图片要求：**
- 推荐比例：5:4
- 推荐尺寸：500px × 400px
- 最大文件大小：128KB
- 格式：JPG 或 PNG

**设计风格：**
- 禅意中式风格
- 金色、红色等传统吉祥色彩
- 莲花、祥云等元素
- 简约现代，留白充足

## 如何制作分享图片

### 方式一：使用提供的 HTML 模板（推荐）

1. 在浏览器中打开 `demo/share-cover-template.html`
2. 选择喜欢的设计变体
3. 使用浏览器截图功能：
   - Chrome/Edge: F12 → Ctrl+Shift+P → "Capture node screenshot"
   - Firefox: F12 → 右键元素 → "Screenshot Node"
4. 保存为 `share-cover.jpg`
5. 使用 [TinyPNG](https://tinypng.com/) 压缩到 128KB 以内
6. 放置到 `frontend/src/assets/` 目录

### 方式二：使用在线设计工具

1. 访问 [Canva](https://www.canva.cn/) 或 [创客贴](https://www.chuangkit.com/)
2. 创建自定义尺寸：500px × 400px
3. 参考 HTML 模板的设计：
   - 背景：金色到红色渐变
   - 主元素：莲花图标 🪷
   - 文案："拜拜" + "AI愿望分析助手"
   - 装饰：祥云、装饰线等
4. 导出为 JPG 格式
5. 压缩到 128KB 以内
6. 放置到 `frontend/src/assets/` 目录

### 方式三：使用 AI 生成工具

使用以下 Prompt 让 AI 设计工具生成：

```
请设计一张微信小程序分享封面图，要求：
- 尺寸：500px × 400px（5:4 比例）

- 主色调：金色、红色渐变
- 元素：莲花图案、祥云装饰
- 文案：
  - 主标题："拜拜 - AI愿望分析"
  - 副标题："让你的愿望更容易实现"
- 布局：居中对称，留白充足
- 氛围：吉祥、温馨、专业
```

## 注意事项

### 合规要求

根据微信小程序规范，分享内容不得包含：

- ❌ 诱导分享（如"分享即可获得"、"不转不是中国人"等）
- ❌ 夸大宣传（如"100%准确"、"包治百病"等）
- ❌ 违规内容（如涉及迷信、赌博等）
- ✅ 客观描述功能和体验（如"愿望分析助手"、"AI 优化许愿"等）

### 图片要求

1. **文件大小**：确保图片小于 128KB，否则可能无法显示
2. **图片路径**：
   - 本地图片路径必须是相对于小程序根目录的绝对路径（如 `/assets/share-cover.jpg`）
   - 在线图片必须使用 HTTPS 协议
   - 在线图片的域名需要在小程序后台配置为合法域名
3. **图片内容**：不能包含违规内容，否则可能被微信拦截

### 标题要求

1. **长度限制**：标题不宜过长，建议 15-25 字
2. **内容合规**：不能包含诱导分享、夸大宣传等违规内容
3. **Emoji 使用**：适度使用 Emoji，不要过多（1-2 个即可）
