## 🎨 标题组合建议

### 组合原则

1. **根据场景选择**：
   - 朋友圈分享：好奇型、社交型
   - 群聊分享：实用型、简洁型
   - 私聊分享：情感型、专业型

2. **根据用户群体选择**：
   - 年轻用户：好奇型、情感型
   - 成熟用户：实用型、专业型
   - 广泛群体：简洁型、社交型

3. **A/B 测试**：
   - 可以准备 2-3 个不同风格的标题
   - 在不同时间段或不同用户群体中测试
   - 根据点击率和转化率选择最优标题

## ⚠️ 注意事项

### 合规要求

所有标题必须符合微信小程序规范：

- ✅ **允许**：客观描述功能和体验
- ✅ **允许**：使用 Emoji 增加吸引力（1-2 个）
- ❌ **禁止**：诱导分享（如"分享即可获得"）
- ❌ **禁止**：夸大宣传（如"100%准确"、"包治百病"）
- ❌ **禁止**：违规内容（如涉及迷信、赌博等）

### 标题长度

- **推荐长度**：15-25 字
- **最短长度**：10 字（保证信息完整）
- **最长长度**：30 字（避免过长影响阅读）

### Emoji 使用

- **推荐数量**：1-2 个
- **使用原则**：与标题内容相关，增强表达
- **避免**：过多 Emoji（超过 3 个）或无关 Emoji

## 🔄 如何更换标题

### 方法一：直接修改代码

在以下两个文件中修改 `title` 字段：

1. `frontend/src/pages/index/index.tsx` (第 411 行)
2. `frontend/src/components/WishEditorModal/index.tsx` (第 435 行)

```typescript
return {
  title: '你的新标题', // 修改这里
  path: sharePath,
  imageUrl: '/assets/share-cover.jpg',
  // ...
}
```

### 方法二：使用配置变量（推荐）

如果经常需要更换标题，建议将标题提取为配置变量：

```typescript
// 在文件顶部定义
const SHARE_TITLE = '我刚测了个愿望分析，准到吓人😱'

// 在 useShareAppMessage 中使用
return {
  title: SHARE_TITLE,
  // ...
}
```

这样修改时只需要改一个地方。

## 📈 测试建议

### A/B 测试方案

1. **准备 2-3 个不同风格的标题**
2. **在不同时间段测试**：
   - 工作日 vs 周末
   - 上午 vs 下午 vs 晚上
3. **记录数据**：
   - 分享次数
   - 点击率（分享卡片被点击的比例）
   - 转化率（点击后注册/使用的比例）
4. **选择最优标题**：根据数据选择表现最好的标题

### 数据指标

- **分享次数**：用户分享的频率
- **点击率**：分享卡片被点击的比例
- **转化率**：点击后注册/使用的比例
- **用户反馈**：用户对标题的评价
