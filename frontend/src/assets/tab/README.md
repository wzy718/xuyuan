# 底部导航栏图标资源

## 文件说明

此目录存放底部 Tab 导航栏的图标，每个 Tab 需要 2 张图标（未选中状态 + 选中状态）。

## 文件命名规范

根据设计，底部有 3 个 Tab，对应图标如下：

### Tab 1：首页（愿望分析）- 经卷图标
- `tab-1-icon.png` - 未选中状态（灰色）
- `tab-1-icon-active.png` - 选中状态（檀金色 #C9A962）

### Tab 2：我的愿望 - 莲花图标
- `tab-2-icon.png` - 未选中状态（灰色）
- `tab-2-icon-active.png` - 选中状态（檀金色 #C9A962）

### Tab 3：个人中心 - 香炉图标
- `tab-3-icon.png` - 未选中状态（灰色）
- `tab-3-icon-active.png` - 选中状态（檀金色 #C9A962）

## 图片要求

- **格式**：PNG（支持透明背景）
- **尺寸**：建议 81x81 像素（微信小程序 Tab 图标标准尺寸）
- **比例**：1:1（正方形）
- **文件大小**：建议控制在 20KB 以内

## 颜色规范

- **未选中状态**：灰色（#999999 或 #CCCCCC）
- **选中状态**：檀金色（#C9A962）

## 使用说明

图标已通过 `app.config.ts` 自动配置，放置文件后重新编译即可生效。

## 生成 Prompt

详细的 Midjourney prompt 请参考：`prd/docs/GUIDE_ILLUSTRATION_PROMPTS.md` 中的"底部导航栏图标 Prompt"部分。

## 注意事项

1. 图标需要简洁清晰，在小尺寸下也能清晰识别
2. 建议使用线条图标或扁平化设计
3. 确保三个图标的视觉重量一致
4. 生成后需要手动调整颜色（未选中/选中状态）
